Pixel API

Pixel API is an APi providing many endpoints for casual applications/bots. It has:

  • Fun Endpoints
    • Fact
      Base URL: https://pixel-api-production.up.railway.app/fact
      Example Response: 
      {"fact":"The word \"dexter\" whose meaning refers to the right hand is typed with only the left hand."}
    • Joke
      Base URL: https://pixel-api-production.up.railway.app/fun/joke
      Example Response: 
      {"joke":"Girls are like internet domain names... the ones I like are already taken."}
    • Riddle
      Base URL: https://pixel-api-production.up.railway.app/fun/riddle
      Example Response: 
      {"riddle":"I am lighter than air but a hundred people cannot lift me. Careful, I am fragile. What am I?","answer":"A bubble"}
    • Quote
      Base URL: https://pixel-api-production.up.railway.app/fun/quote
      Example Response: 
      {"riddle":"I am lighter than air but a hundred people cannot lift me. Careful, I am fragile. What am I?","answer":"A bubble"}
    • Chat Bot
      Base URL: https://pixel-api-production.up.railway.app/fun/chatbot
      Parameters: message[string]
      Example Response: 
      {"riddle":"I am lighter than air but a hundred people cannot lift me. Careful, I am fragile. What am I?","answer":"A bubble"}
    • Advice
      Base URL: https://pixel-api-production.up.railway.app/fun/advice
      Example Response: 
      {"advice":"It's not about who likes you, it's about who you like."}
    • 8ball
      Base URL: https://pixel-api-production.up.railway.app/fun/8ball
      Example Response: 
      {"reply":"Yes â€“ definitely"}
    • Would You Rather
      Base URL: https://pixel-api-production.up.railway.app/fun/wouldyourather
      Example Response: 
      {"option_1":{"question":"Be Cinderella","votes":"357,083"},"option_2":{"question":"Be Ariel","votes":"576,901"}}
    • Crystal Ball
      Base URL: https://pixel-api-production.up.railway.app/fun/crystalball
      Example Response: 
      {"reply":"Outlook good"}
